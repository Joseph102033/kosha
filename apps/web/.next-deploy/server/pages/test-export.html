사고 요약

건설현장 3층 비계 작업 중 안전난간 미설치 구간에서 작업자가 추락한 사고입니다. 안전벨트 미착용 및 작업 전 안전 점검 부재가 주요 원인으로 파악되었습니다. 해당 구간의 안전난간 설치 및 작업자 안전교육 강화가 필요합니다.

근본 원인 분석

  1. 안전난간 미설치: 3층 비계 일부 구간에 안전난간이 설치되지 않았음
  2. 안전벨트 미착용: 작업자가 고소작업 시 필수인 안전벨트를 착용하지 않음
  3. 작업 전 안전 점검 미흡: 작업 시작 전 현장 안전상태 점검이 제대로 이루어지지 않음
  4. 안전교육 부족: 작업자에게 고소작업 안전수칙에 대한 교육이 충분히 제공되지 않음
  5. 현장 감독 소홀: 작업 중 현장 책임자의 안전 감독이 부족하였음
